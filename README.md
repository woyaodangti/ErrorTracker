# ErrorTracker
## 功能简介
用于追踪SpringBoot系统中的错误信息，可以输出到console或者发送到钉钉
![image](https://user-images.githubusercontent.com/13415463/116814072-7ebb3780-ab89-11eb-99d6-4cd25aac44cd.png)
引入该插件，可以让错误告警自动发送到钉钉
